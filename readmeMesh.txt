4MIL
 - Contains file structure to generate 4 million cell mesh for DTMB P4119

	0
	    - contains mesh information generated by checkMesh command
	constant
	    - contains triSurface '.stl' files used in snappyHexMesh and SurfaceFeatureExtract
	    - contains polyMesh of 4 million cell mesh for DTMB P4119
	system
	    - contains blockMeshDict and snappyHexMeshDict used to generate 4 million cell mesh
	    - contains decomposeParDict for parallel mesh generation
	    - contains arbitrary fvSchemes, fvSolution, and controlDict files from Propeller Tutorial - do not use for simulation.
